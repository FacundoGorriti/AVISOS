# Portal de Avisos Interno - Versión Premium

Proyecto hecho con HTML, CSS y JavaScript.

## Incluye

- Diseño premium oscuro.
- Cards modernas con efecto glass.
- Publicación de avisos cortos.
- Orden por llegada.
- Guardado en localStorage.
- Buscador.
- Filtro por categoría.
- Contador de caracteres.
- Eliminar aviso individual.
- Limpiar cartelera.
- Responsive para PC, tablet y celular.
- Footer con firma de Facundo Santiago Gorriti.

## Cómo usar

Abrí `index.html` en el navegador.
